## MCP: Context7

Always use Context7 when you need library or API documentation, code generation, setup steps, or configuration for any library used in this project (Next.js, Supabase, Zod, Tailwind, etc.) without me having to explicitly ask.

Use `resolve-library-id` first to get the correct Context7 library ID, then `query-docs` to fetch relevant documentation.

Key library IDs for this project:
- Next.js: `/vercel/next.js`
- Supabase JS: `/supabase/supabase`
- Supabase SSR: `/supabase/ssr`
- Zod: `/colinhacks/zod`
- Tailwind CSS: `/tailwindlabs/tailwindcss.com`
